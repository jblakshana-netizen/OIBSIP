# Customer Segmentation Analysis

This project focuses on Customer Segmentation Analysis to identify different groups of customers based on their purchasing behavior and characteristics.

- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Google Colab

## Files Included
- Customer_Segment_Analaysis.ipynb
- customer_data.csv
- Project_Report.pdf

## Outcome
This project helped me understand customer behavior, data analysis, clustering techniques, and data visualization.
