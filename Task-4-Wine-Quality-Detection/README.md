# Wine Quality Detection

## Project Overview
This project focuses on predicting the quality of wine based on its chemical properties using machine learning techniques.

## Objectives
- Analyze wine quality data
- Build classification models
- Compare model performance
- Predict wine quality accurately

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Google Colab

## Machine Learning Models
- Random Forest Classifier
- Stochastic Gradient Descent (SGD)
- Support Vector Classifier (SVC)

## Files Included
- Wine_Quality_Detection.ipynb
- winequality.csv

## Outcome
This project improved my skills in machine learning, classification algorithms, model evaluation, and data visualization.
