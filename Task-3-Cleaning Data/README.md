# Data Cleaning

## Project Overview
This project focuses on cleaning and preparing raw data for analysis. Data cleaning is an important step in the data analytics process to improve data quality and accuracy.

## Objectives
- Identify missing values
- Remove duplicate records
- Correct data types
- Handle inconsistent data
- Prepare data for analysis

## Tools Used
- Python
- Pandas
- NumPy
- Google Colab

## Files Included
- Data_Cleaning.ipynb(Notebook Jupyter)
- AB_NYC_2019.csv.zip(Dataset)
- Cleaned_AB_NYC_2019.csv(cleaned Dataset)

## Outcome
This project improved my skills in data preprocessing, handling missing values, removing duplicates, and preparing datasets for analysis.
