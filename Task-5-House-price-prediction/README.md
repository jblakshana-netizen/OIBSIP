# Predicting House Prices with Linear Regression

## Project Overview
This project uses Linear Regression to predict house prices based on various housing features. The goal is to build a machine learning model that can estimate house prices accurately.

## Objectives
- Analyze housing data
- Preprocess and clean the dataset
- Train a Linear Regression model
- Evaluate model performance
- Predict house prices

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Google Colab

## Files Included
- House_Price_Prediction.ipynb
- housing.csv

## Outcome
This project improved my understanding of machine learning, regression models, data preprocessing, and predictive analytics.
